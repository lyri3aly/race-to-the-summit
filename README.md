# Race to the Summit
## A racing game made in PyGame where the user drives through three different seasons in a mountainous region.

<img width="730" height="491" alt="image" src="https://github.com/user-attachments/assets/0d4199e2-e758-45e1-9bc3-ee3d0944b819" />


Download the folder to try the game in action yourself in VSCode or another IDE! (required language version: any with PyGame, I run Python 3.11)



<img width="725" height="505" alt="image" src="https://github.com/user-attachments/assets/2f2ed1f4-074c-448e-80f0-0a7f6374dd69" />

This project/game includes:
- Stopwatch to race yourself and your friends' fastest times
- Three different car colours/variations and three different mountainous season regions
- Car physics (acceleration, friction, etc.)
- Fun camera tracking
